13.1  Restriction on Properties
read-rdf-file accepter-fun [ file [ code ]] [Function]
This function read and parse a file that contains RDF/XML format data, then passes attributes in RDFdecl, parsed XMLDecl, parsed doctypedecl, parsed Comments, and parsed Descriptions to accepter-fun. code is a character code of file. If file has its encoding description in XMLDecl, the direction in file superseeds.
